# IDPA_greenhouse
Dieses Projekt befasst sich mit der IDPA-Arbeit der folgenden Personen:
- Elia Iacovone
- Selina Ackle
- Fabian Mauron
Der Projektinhalt umfasst dabei die Konstruktion, den Aufbau und den Betrieb eines intelligenten Gewächshauses mit Hilfe eines Mikrocontrollers zur automatischen Bewässerung der sich darin befindlichen Pflanzen.
